마크다운 작성 도우미

            
        

        

            
텍스트를 선택한 후 아래 버튼을 클릭하거나, 커서 위치에 문법을 삽입하세요.

            

                
                
                
                
                
                
                
                
                
                
                
                
                
                
                
                
            

            

                
빠른 문법 가이드:

                
# 제목1, ## 제목2, ### 제목3

                
**굵게** 또는 __굵게__

                
*기울임* 또는 _기울임_

                
~~취소선~~

                
[링크 텍스트](URL)

                
![이미지 설명](이미지 URL)

                
- 항목 1
- 항목 2

                
1. 항목 1
2. 항목 2

                
- [ ] 미완료 작업
- [x] 완료된 작업

                
> 인용문

                
```언어\n코드 내용\n```

                
`인라인 코드`

                
--- 또는 *** (수평선)

                
| 제목1 | 제목2 |\n| --- | --- |\n| 내용1 | 내용2 | (테이블)

                
텍스트[^1]\n\n[^1]: 각주 내용 (각주)

            

        

    






    

        

            
JSON 작성 도우미

            
        

        

            
커서 위치에 JSON 구조를 삽입하거나, 선택된 텍스트를 값으로 사용합니다.

            

                
                
                
                
                
                
                
                 
                
            

            

                
JSON 문법 규칙:

                
- 데이터는 이름/값 쌍으로 구성됩니다.

                
- 데이터는 쉼표로 구분됩니다.

                
- 중괄호 {}는 객체를 보유합니다.

                
- 대괄호 []는 배열을 보유합니다.

                
- 모든 속성 이름(키)은 큰따옴표로 묶어야 합니다.

                
- 문자열 값도 큰따옴표로 묶어야 합니다.

            

        

    






    

        

            
XML 작성 도우미

            
        

        

            
커서 위치에 XML 구조를 삽입하거나, 선택된 텍스트를 내용/값으로 사용합니다.

            

                
                
                
                
                
                
                
                
            

            

                
XML 구문 규칙:

                
- 모든 XML 요소는 닫는 태그가 있어야 합니다 (예: <tag>내용</tag>).

                
- 빈 요소는 <tag/> 형식으로 닫을 수 있습니다.

                
- XML 태그는 대소문자를 구분합니다.

                
- 모든 XML 요소는 적절하게 중첩되어야 합니다.

                
- 모든 XML 문서는 유일한 루트 요소가 있어야 합니다.

                
- 속성 값은 항상 따옴표로 묶어야 합니다 (예: attribute="value").

            

        

    






    

        

            
CSV 작성 도우미

            
        

        

            
커서 위치에 CSV 구분자 또는 구조를 삽입합니다. 선택된 텍스트는 적절히 처리됩니다.

            

                
                
                
                
                
                
                
                
            

            

                
CSV 작성 규칙:

                
- 각 레코드(행)는 새 줄로 구분합니다.

                
- 각 필드는 쉼표(,)로 구분합니다.

                
- 필드 내에 쉼표, 줄바꿈 문자, 또는 큰따옴표가 포함된 경우, 해당 필드는 큰따옴표(")로 감싸야 합니다.

                
- 필드 내에 큰따옴표 자체가 포함된 경우, 두 개의 큰따옴표("")로 이스케이프 처리합니다.

                
예: 이름,나이,"주소, 상세","메모(""중요"")"
